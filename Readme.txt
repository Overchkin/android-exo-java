Exercice:
On va créer un zoo avec différents animaux. Chaque animal a une race un nom,et fait quelque 
chose comme bruit (un chien fait hau, haut par exemple).

1- Créez la classe animal.

2- Créez une méthode pour la classe animal qui écrit sur l'écran la présentation d'un anima:
son nom sa race et ce qu'il fait comme bruit.

3- Créez les classes dérivées pour les différentes animaux: chien, chat, vache etc.

4- Créez une classe ZooMain qui contient la méthode main.

5- Créez les animaux du zoo dans une variable static de la classe ZooMain en forme
d'un tableau. Les cases du tableau correspondent aux cages du zoo.

6- Créez un chien et deux chats dans votre zoo et faites les présenter.

7- Ajouter une interface pour la méthode de présentation et modifier le code.
